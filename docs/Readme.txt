To create the documentation:
1. pip install sphinx
2. Create a empty folder named 'build' inside 'docs' folder
3. On your command line cd to the 'docs' folder
4. On your command line write 'make html'
5. To check how it looks goto: '..\docs\build\html\index.html'

To delete the documentation:
1. On your command line cd to the 'docs' folder
2. On your command line write 'make clean'
